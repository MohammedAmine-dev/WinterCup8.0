# WinterCup 8.0

This repository is a small record of our run at the national contest WinterCup 8.0. My team, kabout_beloté, competed here and stayed consistently in the mix across the event.

The folder mainly contains contest solutions and practice code in Python and C++. It was built the way these contests usually are: fast, practical, and focused on getting answers under pressure.

## About the contest

WinterCup 8.0 was a national programming contest that rewarded consistency as much as speed. For us, the important part was keeping a steady rhythm, solving what we could cleanly, and staying competitive throughout.
